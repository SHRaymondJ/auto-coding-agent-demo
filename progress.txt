## 2026-02-15 - Task 39: 导航栏行为优化 - 滚动隐藏/显示

### What was done:
- Header 组件新增滚动方向检测逻辑（useCallback + useRef 追踪 lastScrollY）
- 向下滚动超过 5px 阈值时隐藏导航栏（-translate-y-full）

- 移动端菜单打开时不隐藏导航栏（isHidden && !isMobileOpen）
- 使用 CSS transition-all duration-300 实现平滑滑入/滑出动画

### Testing:
- npm run lint 通过（0 errors）
- npm run build 通过（29 页面全部静态生成）
- 浏览器测试：首页和子页面（/about）均验证滚动隐藏/显示/停止恢复三种行为正常

### Notes:
- 未使用 framer-motion，CSS transition 已足够实现平滑的掉落效果
- 5px 阈值防止微小抖动触发状态切换
